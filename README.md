# Homebrew Tap for TermIDE

This is the official [Homebrew](https://brew.sh) tap for [TermIDE](https://github.com/termide/termide) - a cross-platform terminal IDE, file manager and virtual terminal.

## Installation

```bash
brew tap termide/termide
brew install termide
```

## Upgrade

```bash
brew upgrade termide
```

## Uninstall

```bash
brew uninstall termide
brew untap termide/termide
```

## About TermIDE

TermIDE is a terminal-based IDE featuring:

- Syntax highlighting for 15+ programming languages
- Smart file manager with git integration
- Integrated virtual terminal with full PTY support
- Multi-panel layout system
- 12 built-in themes
- Multi-language support (9 languages)

For more information, visit the [main repository](https://github.com/termide/termide).
